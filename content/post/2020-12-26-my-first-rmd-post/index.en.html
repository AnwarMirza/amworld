I am using python the draw sine and cosine curves. The code and the results are here.


import numpy as np
import matplotlib.pyplot as plt
t = np.arange(0, 5*np.pi, 0.1)
y = np.sin(t)
z = np.cos(t)
plt.plot(t,y)
plt.plot(t,z)
